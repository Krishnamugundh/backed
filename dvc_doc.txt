to store individual files into seperate dvc remotes, follow the steps;

1.) dvc remote add --remote  <remote_name>  <url> --local
2.) dvc config cache.local  <remote_name>
3.) dvc add <file_name>
4.) dvc push --remote  <remote_name>

            OR

1.) dvc remote add --remote <remote_name> <url> 
2.) dvc add file_1
3.) dvc add file_2
4.) dvc push --remote <remote_name> file_1.dvc
5.) dvc push -r <remote_name> file_2.dvc

    for pulling them;
6.) dvc pull -r <remote_name> file_1.dvc
7.) dvc pull --remote <remote_name> file_2.dvc


to revert or go-back to the previous version of your commited file.

1.) git checkout HEAD^1 <file_name(.dvc file if its dvc tracked)>

to again come to the last commit;

1.) git checkout -- <file_name(.dvc file if its dvc tracked)>
